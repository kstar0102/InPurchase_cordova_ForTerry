___

### CONSUMABLE

• **CONSUMABLE** = ``"consumable"``

• **NON\_CONSUMABLE** = ``"non consumable"``

• **NON\_RENEWING\_SUBSCRIPTION** = ``"non renewing subscription"``

Type: An non-renewing subscription

___

### PAID\_SUBSCRIPTION

• **PAID\_SUBSCRIPTION** = ``"paid subscription"``
